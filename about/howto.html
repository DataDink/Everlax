How To

Using Everlax is very easy. All you need to do is tell Everlax what direction and how fast and it does the rest. Everlax has two ways to configure an animation:

Inline Data-Attributes
  1. <div style="background: url('image1.png'), url('image2.png')"
  2.      data-everlax-duration="20s"
  3.      data-everlax-direction="90"></div>
  5. <script type="text/javascript">
  6.     $('div').everlax();
  7. </script>
Object Configuration
  1. <div style="background: url('image1.png'), url('image2.png')"></div>
  3. <script type="text/javascript">
  4.     $('div').everlax({direction: 0, duration: '10s'});
  5. </script>

Everlax will examine the background images styled on the configured element, calculate separate start and end points for each, and finally create a CSS3 Keyframes animation for your element.

The Everlax element must be styled with at least one background image in order for any effect to be visible. Everlax supports multiple background images on a single element, but the direction and duration configured on the element applies to the whole animation and not individual images.

  1.     .my-everlax-element {
  2.         background: url('my-image1.png'), url('my-image2.png');
  3.     }
Support & Limitations:
  • Everlax has some limited support for background positioning and sizing.
  • It does not support element size changes.
  • Styles applied by Everlax must be removed from a resized element and the plugin re-applied.
  • Angles are approximate as Everlax will make minor adjustments to keep end animation parameters reasonable.
